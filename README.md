# NUSHIR AGROTEC AND FARMS PRIVATE LIMITED Website

This project is a premium corporate website for NUSHIR AGROTEC AND FARMS PRIVATE LIMITED.
The design is modern, responsive, and built using HTML5, CSS3, and vanilla JavaScript.

## Files

- `index.html` - Main website page
- `style.css` - Custom styles for layout, animations, and responsive behavior
- `script.js` - Interactive behaviors: smooth scrolling, testimonials slider, FAQ accordion, counters, progress bar, loading animation
- `assets/favicon.svg` - SVG favicon placeholder
- `images/` - Directory for portfolio and certificate assets

## How to run

1. Open `index.html` directly in your browser, or
2. Start a local server in the project folder and navigate to `http://localhost:8000`

## Notes

- No Bootstrap, React, or Tailwind is used.
- The site focuses on software development, app development, and digital business solutions.
- The certificate section is prepared for a future image insert.
